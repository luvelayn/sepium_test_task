# Sepium Test Task

Тестовое задание — карточки товаров с слайдером, модальным окном и адаптивной версткой.

## Стек

- HTML, CSS, JavaScript
- Fancyapps — слайдер и модальное окно (Carousel, Fancybox)
- ESLint + Prettier

## Запуск

Проект использует ES Modules, поэтому требует локальный сервер (открыть index.html напрямую не получится).

Через VS Code: расширение **Live Server** → ПКМ на index.html → Open with Live Server

Или через Node.js:
```bash
npx serve .
```

## Структура проекта
```
├── common.blocks/          # Блоки проекта

│   │   └── card.js
│   └── products/           # Блок списка товаров
```

## Реализовано

- Адаптивная сетка карточек (десктоп / мобайл)
- Слайдер: автовоспроизведение на мобайл, кнопки на десктопе
- Анимации и ховер-эффекты для интерактивных элементов
- Модальное окно с галереей через Fancybox
- Переключение модификации материала
- Лайк с обращением к API
- Клик по карточке открывает страницу товара
